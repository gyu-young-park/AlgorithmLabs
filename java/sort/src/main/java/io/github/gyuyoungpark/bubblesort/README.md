# BubbleSort
거품이 생겼다가 빠지듯이 정렬하는 방법이라 bubble sort라 하는데, 알고리즘 방식은 매우 단순하다.

## 원칙
나와 내 오른쪽을 비교해서 내가 더 크다면 오른쪽으로 간다. 이렇게 비교하다보면 맨 뒤는 가장 큰 값이 되어있을 것이다.

1. 인덱스 i와 i + 1의 값을 비교한다.
2. 인덱스 i에 있는 값이 더 크다면 i + 1과 자리를 바꾼다.
3. 만약 인덱스 i에 있는 값이 더 작다면 자리를 바꾸지 않는다.
4. i = i + 1 가 된다.

옆에 친구들과 비교를 하다보니 O(n^2)이다. 옆에 친구와 비교해서 바꾸냐 안바꾸냐를 결정하기 때문에 값의 순서가 유지되는 stable 정렬 특징이 있다.
