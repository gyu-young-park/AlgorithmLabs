# 선택 정렬

## 원리
전체 array를 보고 가장 작은 녀석은 하나 찾자, 그리고 그 녀석을 가장 앞에 두도록 하자. 이 다음에는 두 번째로 작은 녀석을 찾고, 두번째에 넣자, n번째 작은 녀석을 찾았을 때 그 녀석은 이미 n번째 자리에 있어야 한다.

1. 전체 array를 한 번 스캔 후 가장 작은 값의 인덱스 i를 찾음
2. 인덱스 0과 인덱스 i 값을 바꾼다.
3. 인덱스 0을 제외하고 나머지 중에 가장 작은 값을 찾아서 반복한다.

O(n^2)이 걸리며, 옆에 친구와 비교하는 것이 아니라 전체를 비교하여 swap하기 때문에 unstable하다는 특징이 있다.
